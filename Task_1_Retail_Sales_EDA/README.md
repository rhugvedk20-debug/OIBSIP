# Retail Sales EDA Project

## Objective
To analyze retail sales data and extract business insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Monthly and quarterly sales trends
- Top-selling products
- Category-wise revenue analysis
- Customer segment behavior
- Correlation between variables

## Insights
- Sales show seasonal patterns
- Certain products dominate revenue
- Discounts impact profitability
- Customer segments differ in contribution

## Conclusion
This project helps understand business performance and supports data-driven decision-making.
